This field creates a drop down of current user's orders. Contact Form7 form will be more useful to use for support purpose.You just have to install this plugin to add a new field type in Contact form 7 fields.For details usages guide please check [the documentation](https://codeboxr.com/product/contact-form-7-woocommerce-orders/)== Installation ==This section describes how to install the plugin and get it working.e.g.1. Upload `contact-form-7-wooorders` to the `/wp-content/plugins/` directory2. Activate the plugin through the 'Plugins' menu in WordPress3.
